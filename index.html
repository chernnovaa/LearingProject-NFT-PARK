Each token has a number of parameters:

Tree Name

Each sapling (tree) has a unique name, the name can be changed after purchasing a token. C02 tokens will be burned to change the name

Image

We will update the image every 6 months until it takes its final shape. Each image will be engraved with the exact coordinates of your tree

Tree age

The age of the tree will be counted from the moment of planting the seedlings in the ground and denoted by years and days

SEQ tokens

Each tree (NFT token) generates a certain number of ERC20 tokens per day. The number of tokens generated = the amount of C02 absorbed by the tree

Tokenomics

Each NFT tree token generates a certain number of ERC20 standard tokens (SEQ Token).

  • Each tree generates 1,000 SEQ tokens per day

  • Every month the number of generated tokens increases by 100

  • You must burn 30,000 tokens to change the name of the tree

The owner of a SEQ token may purchase items to enoble his tree. For example, you can buy Christmas decorations, order a drone shot of your tree, and more.
